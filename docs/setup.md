# Getting set up

## Prerequisites

- Node.js installed on your system.
- npm package manager, npm is automatically installed with Node.js.

## Steps

1. **Clone the project repository from GitHub**
2. **Install VSCode extension [Prettier ESlint](https://marketplace.visualstudio.com/items?itemName=rvest.vs-code-prettier-eslint)**
3. **Navigate to the project directory**
4. **Install project dependencies:**

```sh
yarn install
```

5. **Start the development server:**

```sh
yarn dev
```

6. **Open your web browser and go to the following address:** [http://localhost:3000](http://localhost:3000)

You should now see your project running in your local browser.

# Build the application for production

1. **Compiles the application for production:**

```sh
yarn build
```

2. **Starts the application in production mode:**

```sh
yarn start
```
